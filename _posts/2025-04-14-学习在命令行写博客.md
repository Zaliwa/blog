# 使用命令行操作 GitHub Blog

## 安装 GitHub CLI

1. **下载安装**
   访问 [GitHub CLI 官方页面](https://cli.github.com/)，选择适合你操作系统的安装包进行安装。

2. **验证安装**
   安装完成后，在终端运行以下命令验证是否安装成功：
   ```bash
   gh --version
   ```

## 配置 GitHub CLI

1. **登录 GitHub**
   在终端运行以下命令进行登录：
   ```bash
   gh auth login
   ```
   按照提示操作，选择登录方式（如通过浏览器登录）并完成身份验证。

2. **配置 Git**
   设置 Git 用户名和邮箱（建议使用 GitHub 注册邮箱）：
   ```bash
   git config --global user.name "你的用户名"
   git config --global user.email "你的邮箱"
   ```

## 创建和管理 GitHub Blog

1. **创建新仓库**
   ```bash
   gh repo create your-blog-name --public
   ```
   这将创建一个名为 your-blog-name 的公共仓库。

2. **克隆仓库到本地**
   ```bash
   gh repo clone your-blog-name
   ```
   或者使用 Git 命令：
   ```bash
   git clone https://github.com/your-username/your-blog-name
   ```

3. **添加博客内容**
   - 在克隆的仓库目录中，创建 Markdown 文件（如 post1.md）作为博客文章。
   - 编辑文件内容，添加标题、正文等。

4. **提交并推送内容**
   ```bash
   git add .
   git commit -m "Add new blog post"
   git push origin main
   ```

## 管理博客仓库

1. **查看仓库状态**
   ```bash
   gh repo view your-blog-name --web
   ```
   这将打开浏览器中的仓库页面。

2. **列出仓库中的文件**
   ```bash
   gh repo view your-blog-name --web --show-files
   ```
   这将显示仓库中的所有文件。

## 配置 SSH（可选）

如果你希望使用 SSH 方式连接 GitHub，可以生成 SSH 密钥并添加到 GitHub：

1. **生成 SSH 密钥**
   ```bash
   ssh-keygen -t rsa -b 4096 -C "你的注册邮箱"
   ```
   按提示操作，生成密钥对。

2. **添加 SSH 密钥到 GitHub**
   - 打开生成的公钥文件（默认路径为 ~/.ssh/id_rsa.pub），复制内容。
   - 登录 GitHub，进入 Settings > SSH and GPG keys，点击 New SSH Key，粘贴公钥并保存。
